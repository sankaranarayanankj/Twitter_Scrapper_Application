Installation of Python and Dependencies
1)Install Python Version 3.10.0-amd64
2)Install all the pip dependencies in the Requirements.txt file


How to run the Scraper with Twitter 
Run the save_session.py function to get the twitter login session.
1) python save_session.py 
2) Allow the chromium session as administor.
3) Login to the Twitter/X for the twitter_session for the Login Creditional and Information.
4) twitter_session.json is created from the Creditional.

Run the twitter_scraper_with_login.py to get the given hashtags.
1) python twitter_scraper_with_login.py
2) Program runs to collect the hashtags
3) Run all the hastags.
4) After the Program completed, "indian_market_tweets.csv" is created.
